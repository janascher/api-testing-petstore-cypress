Cypress
- Framework automação para API e Web (famoso), em JAvaScript
- Não requer Framework de Teste de Unidade (egine interno)
- Executa os testes dentro do navegador (browser) --> maior desempenho
- Alto desempenho (até 10x, comparado com outros frameworks) --> Apenas em aplicações com arquitetura SPA (Single Page Application)
- Concorrentes principais: Selenium WebDriver, Playwright e Robot Framework
